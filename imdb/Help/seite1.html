How to use

Search by Annotations


Enter keywords/tags and press search button to get respective images. The annotations were assigned manually to the pictures in the database.


Search by Image

You can upload a image and search for similar images by clicking on search button.
By default, Google inception-v3 algorithm is used for feature extraction and cosine is used for distance computation.
User can play with the advanced search by changing the feature extraction algorithms, distance algorithms and number of clusters.

Feature algorithm

There are three feature extraction algorithms:
1. Inception-v3
2. VGG 16
3. VGG 19

Number of clusters slider can be used to filter results as per accuracy. The more the number of clusters, better the accuracy and fewer results.

Distance algorithm

User can select distance algorithms like Euclidean and cosine distance where cosine similarity is advantageous because even if the two similar images are far apart by the Euclidean distance because of the features(Image embedding), colour, Size etc.
